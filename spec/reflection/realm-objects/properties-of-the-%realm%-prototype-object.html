The value of the [[Prototype]] internal slot of the %Realm% prototype object is the standard built-in Object prototype object (19.1.3 ). The %Realm% prototype object is an ordinary object. It does not have a [[Realm]] internal slot .

The initial value of %Realm%.prototype.constructor is the built-in %Realm% constructor.

The initial value of the @@toStringTag property is the string value "Realm ".

This property has the attributes { [[Writable]]: false, [[Enumerable]]: false, [[Configurable]]: true }.

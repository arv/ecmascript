The abstract operation CreateListIterator with argument list creates an Iterator (25.1.2 ) object whose next method returns the successive elements of list. It performs the following steps:

  1. Let iterator be the result of ObjectCreate (%ObjectPrototype%, ([[IteratedList]], [[ListIteratorNextIndex]])).
  2. Set iterator’s [[IteratedList]] internal slot to list.
  3. Set iterator’s [[ListIteratorNextIndex]] internal slot to 0.
  4. Define ListIterator next (7.4.7.1 ) as an own property of iterator.
  5. Return iterator.

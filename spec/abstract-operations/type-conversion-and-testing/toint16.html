The abstract operation ToInt16 converts its argument to one of 216 integer values in the range −32768 through 32767, inclusive. This abstract operation functions as follows:

  1. Let number be the result of calling ToNumber on the input argument.
  2. ReturnIfAbrupt (number).
  3. If number is NaN, +0, −0, +∞, or −∞, return +0.
  4. Let int be sign (number) × floor (abs (number)).
  5. Let int16bit be int modulo 216 .
  6. If int16bit ≥ 215 , return int16bit − 216 , otherwise return int16bit.
